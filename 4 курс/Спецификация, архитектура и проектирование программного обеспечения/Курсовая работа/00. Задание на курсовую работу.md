В рамках дисциплины необходимо выполнить курсовую работу. Задания, выполненные в рамках ЛР должны быть использованы в качестве основы для соответствующих разделов.  
Для защиты курсовой работы должно быть выполнено и сдано РПЗ, демонстрация выполненной работы с презентацией.  
## Разделы расчётно-пояснительной записки
1. Сравнительный анализ отечественных и зарубежных аналогов проектируемой системы
2. Выбор технологии, среды и языка программирования
3. Анализ процесса обработки информации, выбор структур данных для ее хранения, выбор методов и алгоритмов решения задачи
4. Разработка спецификаций проектируемой системы
	- 4.1. Построение диаграммы вариантов использования
	- 4.2. Построение диаграммы классов
	- 4.3. Построение диаграммы последовательностей системы
	- 4.4. Построение диаграммы деятельностей вариантов использования
5. Проектирование программного обеспечения
	- 5.1. Проектирование структуры системы и построения диаграмм пакетов
	- 5.2. Проектирование пакета "..."
		- 5.2.1 Исходная диаграмма классов
		- 5.2.2. Диаграмма последовательностей взаимодействия объектов
		- 5.2.3. Диаграмма кооперации
		- 5.2.4. Уточненная диаграмма классов
		- 5.2.5. Детальная диаграмма классов
	- 5.3. Модульная структура проекта
	- 5.4. Построение диаграммы компонентов
	- 5.5. Построение диаграммы размещения
6. Тестирование
