## Операторы
- Арифметические: +, -, /, \*, %, ++, --
- Побитовые: &, |, ^, ~, <<, >>
- Присваивание: =, +=, -=, \*=, /=, %=, ...
- Сравнение: \==, !=, <, >, <=, >=
- Логические: ||, &&, |
- null-условные операторы и проверки типа и выражения приведения: ??, ?., |., ?\[\], is, as, ?:
## Условные конструкции и циклы
Условные конструкции:
- if, else
- switch, break
- goto case
- return
- throw
  
Циклы:
- for, break
- foreach, continue
- while
- do while
