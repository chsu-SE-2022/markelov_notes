1. Создаем в Блокноте файл `test.txt`
2. В Проводнике изменяем расширение на `test.htm`
  
Структура web-страницы:
```html
<html>
	<head>
		<title> какой-то текст </title>
	</head>
	
	<body>
		какой-то текст
	</body>
</html>
```
Пример:
```html
<html>
	<head>
		<title>Первая страница</title>
	</head>
	
	<body>
		Привет!
	</body>
</html>
```
