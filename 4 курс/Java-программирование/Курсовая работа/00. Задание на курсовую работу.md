## Тема: "Программирование на языке Java"
**Spring Framework** (или коротко **Spring**) - универсальный фреймворк с открытым исходным кодом для Java-платформы.  
Авторы Spring решили предоставить разработчикам некоторые утилиты, которые автоматизируют процедуру настройки и ускоряют процесс создания и развертывания Spring-приложений, под общим названием Spring Boot.  
**Spring Boot** - это полезный проект, целью которого является упрощение создания приложений на основе Spring. Он позволяет наиболее простым способом создать web-приложение, требуя от разработчиков минимум усилий по его настройке и написанию кода.  
**Vaadin** - свободно распространяемый фреймворк для создания веб-приложений, разрабатываемый одноимённой финской компанией. Одна из наиболее значимых функций в Vaadin - использование Java как единственного языка программирования при создании веб-приложений и веб-контента.  
Ваша задача использовать лекции по Spring Framework и Vaadin, а также ознакомиться с фреймворками на сайте http://spring-projects.ru/ или на оригинальных сайтах https://spring.io/, https://vaadin.com/.  
Далее необходимо разработать Java-приложение с подключением классов фреймворка Spring c использованием Spring Boot и Vaadin.
## Общие требования к программному обеспечению для разработки технического задания
- В одной из СУБД по вашему выбору спроектировать и создать базу данных предметной области, выбранной согласно варианту. Количество сущностей в базе данных должно быть **не менее трех (лучше пять!)**
- С использованием Spring Boot и Vaadin создать веб-приложение, работающее с созданной базой данных
- Web-приложение должно позволять просматривать таблицы БД, производить добавление и удаление записей, а также поиск и фильтрацию данных
- По данным разработанной БД средствами фреймворка Vaadin создать дэшборд
- Интерфейс приложения должен быть дружественным. Графические изображения, иконки и т. д. приветствуются. Для создания интерфейса использовать фреймворк Vaadin
- Работа всех функций должна быть проверена и результаты проверки оформлены протоколом тестирования
